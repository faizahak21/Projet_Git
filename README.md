# TP Git – Mini projet JavaScript

Projet réalisé dans le cadre du TP Git afin de pratiquer les bonnes pratiques de gestion de versions avec **Git** et **GitHub**.

## Objectifs pédagogiques

- Utilisation des **commits** et de leur historique  
- Travail avec des **branches feature**  
- Compréhension des opérations **merge** et **rebase**  
- Utilisation du **rebase interactif** pour nettoyer l’historique  
- Travail collaboratif via **Pull Request**  
- Gestion d’un **dépôt distant GitHub**

## Fonctionnalités du projet

- Calcul d’une addition en JavaScript  
- Affichage du résultat dans la console du navigateur  
- Création d’un objet `person` avec une méthode de présentation  
- Affichage du **jour de la semaine**  
- Amélioration du **format d’affichage console**

1. Ouvrir le fichier `index.html`

## Bonnes pratiques Git respectées

- Commits au format **conventional commits** (`feat`, `refactor`, `docs`, …)  
- Nommage uniforme des branches (`feature/...`)  
- **Pull Request** avant chaque fusion dans `main`  
- Nettoyage de l’historique avec **rebase interactif**  
- Une seule branche principale **main** à la fin du projet
